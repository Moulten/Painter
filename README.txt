Программа позволяет отрисовывать круги и прямоугольники по их местоположению и размерам, 
а также удалять их в обратном порядке.

Цвета выбираются автоматически.



Доработки:

 - Проверка ввода с выводом окна с сообщением о неправильных параметрах

 - Добавление элипса, треугольника и многоугольника

 - Поворот фигур

 - Замена вектора параметров в структуре FigureParams

 - Добавление Класса окна "Дискриптор" для вывода текста, с возможностью менять
   его в ходе программы
